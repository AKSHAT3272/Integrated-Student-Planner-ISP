## Usage

```sh
1. Run pip install -r requirements.txt (Python 2), 
or pip3 install -r requirements.txt (Python 3)
2. Read *dbconfigReadme.txt* to configure database.
3. Run *flaskSettings* shell script to run the app. 

```
